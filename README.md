## What is this?

A personal collection of layered PSD files for graphic design. These templates cover dark, aesthetic, and themed styles. Each file is a standalone template with named layers and adjustable elements.

---

## Templates

| File | Style |

| `Aesthetic Grey.psd` | Greyscale aesthetic layout |
| `Aesthetic Shade.psd` | Shaded aesthetic variant |
| `Aesthetic.psd` | Clean aesthetic base template |

| `Apocalypse.psd` | Post-apocalyptic theme |
| `Doom.psd` | Dark, doom-themed design |
| `FAL.psd` | Custom themed template |
| `RedHawk.psd` | Red and black hawk theme |

---

## How to Use

1. Open Adobe Photoshop (CS6 or newer recommended)
2. Open the `.psd` file you want to edit
3. Edit the layers panel on the right to modify text, colors, or images
4. Export via **File > Export > Export As** (PNG, JPG, etc.)

---

## Requirements

- Adobe Photoshop CS6 or newer
- Any fonts referenced in a template must be installed on your system
